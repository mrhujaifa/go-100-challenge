# 🚀 100 Go (Golang) Programming Challenges

Welcome to the **Go 100 Challenge**! This repository features 100 structured Go programming challenges focused on core logic, problem-solving, and building backend-ready developer skills.

## 📁 Project Structure
Each folder represents an independent project challenge containing complete logic:
- `p001_temperature_converter`: Basic control flow & math inputs.
- `p002_atm_security`: Pin validation logic and CLI workflows.
- `p012_url_shortener`: Custom character mappings, pointer slices, and local data persistence.

## 🛠️ How to Run Locally
1. Clone this repository: `git clone https://github.com`
2. Navigate to a challenge: `cd p012_url_shortener`
3. Run the entry file: `go run main.go`
